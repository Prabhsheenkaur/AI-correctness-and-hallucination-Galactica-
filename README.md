🍷 AI Correctness & Hallucination Detection (Galactica)

This project focuses on evaluating AI model correctness and detecting hallucinations using machine learning techniques. It analyzes predictions made by AI models and compares them with actual outcomes to measure reliability and performance.


📌 Project Overview


Artificial Intelligence models can sometimes generate incorrect or fabricated outputs (hallucinations). This project aims to:


Evaluate model accuracy

Detect hallucinated outputs

Analyze feature importance

Handle class imbalance issues

Compare multiple models for performance

✔️ Feature importance analysis

📊 Evaluation Metrics

Accuracy

Precision

Recall

F1-score

Confusion Matrix

⚠️ Problem Addressed

Class Imbalance


Some classes in the dataset are underrepresented, which can lead to biased predictions.


Hallucination Detection


The system identifies when the model predicts unrealistic or incorrect outputs compared to real data.
